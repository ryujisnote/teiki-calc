# teiki-calc

通勤定期券の「元が取れる日数（往復回数）」を可視化するシンプルなWebツールです。
運賃と定期代を入力するだけで、月間の損益分岐点を瞬時に算出します。
このコードと README.md は Gemini にて作成しました。

## 🚀 特徴

* **Serverless**: HTML/JSのみで完結。外部サーバーへの通信なし。
* **Lightweight**: 1ファイルの静的コンテンツ。
* **Responsive**: GitHub Pagesにデプロイすれば、出先でもスマホから確認可能。

## 🛠 使い方

1. ブラウザで `index.html` を開く（または GitHub Pages の URL にアクセス）。
2. 「区間」「片道運賃」「各定期代」を入力。
3. 「計算する」ボタンをクリックして結果を表示。

## 📝 計算ロジック

各月（1, 3, 6ヶ月）において、1ヶ月あたり何往復すれば元が取れるかを計算します。

$$\text{必要往復回数/月} = \frac{\text{定期券代}}{\text{片道運賃} \times 2 \times \text{月数}}$$
